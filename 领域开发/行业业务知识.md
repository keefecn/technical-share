## 新零售

新零售是一个泛行业，包括电商、消费品零售等行业的综合集合体。典型特征如下：

1. 以消费者为中心。
2. 线上线下融合 OMO
3. 互联网技术与数据技术广泛应用



表格 新零售的数据应用

| 数据应用           | 所属行业 | 数据方案                                   | 数据成果                    | 应用效果       |
| ------------------ | -------- | ------------------------------------------ | --------------------------- | -------------- |
| 基于场景的精准营销 | 互联网   | 收集用户数据                               | 营销建议                    | 提高营销成功率 |
| 线下门店智能发券   | 新零售   | 关联商品分析、顾客购物偏好分析、时效优惠券 | 针对顾客 的推荐商品或优惠券 | 销售额提升     |
| 线下门店竞品分析   |          |                                            | 商品定价建议                |                |
